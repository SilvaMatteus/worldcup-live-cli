World Cup live on terminal.

## Running

Install requirements.

`pip install -r requirements.txt`

Then

`python worldcup_live.py`

**Available commands:**

- today
- tomorrow
- c _clear screen_
- h _show help_

## ROADMAP

**Match information:**

- Attempts on goal;
- Attempts on target;
- Red cards;
- Yellow cards.

**Groups information:**

- Data on screen;
- Cosmetics.
